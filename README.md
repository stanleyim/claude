📊 AI Stock Radar PRO
(DART 기반 한국 주식 데이터 분석 시스템)
🔷 개요
AI Stock Radar PRO는
한국 주식시장(KOSPI / KOSDAQ)의 **실시간 시장 데이터 + 기업 재무데이터(DART)**를 결합하여
유망 종목을 자동으로 선별하고 성과를 검증하는 AI 기반 투자 분석 시스템입니다.
단순 기술적 분석이 아니라
기업의 “실적 + 시장 흐름 + 수급 + 기술적 신호”를 동시에 분석하는 구조입니다.
⚙️ 시스템 핵심 구조
이 시스템은 5단계 완전 자동화 구조로 작동합니다.
1️⃣ 시장 + 기업 데이터 수집 (DART 통합)
KOSPI / KOSDAQ 전체 종목 데이터 수집
주가, 거래량, 변동성 분석
📌 DART API 기반 재무 데이터 수집
DART 데이터 포함 항목:
매출 성장률
영업이익 / 순이익
ROE / ROA
부채비율
현금흐름 구조
👉 “기업의 체력”까지 분석하는 구조
2️⃣ AI 종목 분석 및 점수화
각 종목은 다음 요소로 종합 평가됩니다:
📊 기술적 요소
이동평균
모멘텀
거래량 급증
추세 강도
💰 재무 요소 (DART)
실적 성장성
수익성
재무 안정성
기업 지속 가능성
⚠️ 리스크 요소
과열 종목 필터링
비정상 급등 제거
👉 최종적으로 0~100점 AI Score 생성
3️⃣ 종목 선정 (Top 1~10)
AI Score 기반으로:
상위 10개 종목 자동 선정
순위 기반 가중치 적용
투자 후보군 생성
4️⃣ 성과 추적 시스템
선정된 종목은 자동으로 추적됩니다:
매수 시점 기록
D+1 / D+3 / D+5 / D+10 / D+20 수익률 계산
실제 시장 기반 성과 분석
👉 “추천이 맞았는지”가 숫자로 검증됨
5️⃣ 통계 및 성과 분석
전체 결과는 자동 통계화됩니다:
전체 적중률
평균 수익률
순위별 성과
점수 구간별 정확도
월별 성과 추이
